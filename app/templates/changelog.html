Changelog

23 Sept 2018 - Some overdue fixes
  • Fixed non-player deaths counting as player deaths. This will affect matches previously parsed as well.
  • The map play percentage on the index now only shows data for the last 30 days of matches played.
7 Jun 2018 - Wow, something changed?
  • Added a view for population stats grouped by hour over the last 30 days under global stats.
15 Jan 2018 - Oh god so many bugs
  • Charting library has been moved to something I find to be cleaner. As a result, pie charts are gone. Because they suck. They've been replaced by a bar percentage chart... thing. Go look at it.
  • Fixed population snapshots not parsing.
15 Jan 2018 - Belabored Update Edition
  • Lots of behind-the-scenes changes.
  • Fixed some malf info problems, including them always being shown as winners when they didn't earn it.
  • Since so much changed this time, I don't doubt there'll be lots of bugs.
  • Next on the list: Better charting. After that, per-gamemode global stats maybe.
3 Nov 2017 - Lynch Sood Edition
  • Hello, new changelog page! I'm not going to even pretend to have a proper versioning system, so here's dates instead.
  • Added a hover tooltip to the death counter on match pages.
  • Due to some optimizations to how data is stored, matches before today have been lost because the migration would honestly be painful to perform. This shouldn't happen again for some time.
  • Current plans involve a better view on mobile and adding display of things that have been gathered for a long time but never actually shown anywhere (uplink items!)
  • If you have something you want to see added, check the issues list on GitHub (link in the top right) to see if it's in the works, then either nag me about it on that issue, or make a new one to nag about one that isn't listed.
  • That includes things as simple as "X should be more visible!"
